# Live Coding Session 
Code-based performances by Jens Vetter, 45 grados (Emilio Yáñez & Víctor Badillo) and Nicolás Henao Bonnet

## Location: OKP courtyard
## Time: 04.07.2026, 15:00 – 18:00
Welle by Jens Vetter is a web-based live coding environment that can be used to explore pattern-based electronic music, to quickly try out rhythms and song ideas, to control external MIDI devices and to store, recall and record the musical sessions. Electroceramics by 45 grados (Emilio Yáñez & Víctor Badillo) is conceived as a small live presentation / performance combining live coding with sound devices developed during this semester.The work explores the sonic possibilities of objects produced using traditional Mexican ceramic-making techniques and circuit bended toy instruments.
